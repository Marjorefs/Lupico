# Lúpico

Aplicativo desenvolvido para auxiliar pacientes com Lúpus Eritematoso Sistêmico (LES) no acompanhamento de sua rotina de cuidados diários.

## Funcionalidades

- Cadastro de usuários
- Login
- Perfil
- Registro e histórico de sintomas
- Registro e histórico de medicamentos
- Registro de cuidados
- Upload de receitas em PDF
- Upload de exames em PDF

## Tecnologias utilizadas

- React Native
- Expo
- Supabase
